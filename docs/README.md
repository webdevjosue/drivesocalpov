## 🎯 **Key Resources**

### **🚀 Quick Links**
- **Development Server**: `npm run dev` → `http://localhost:3000`
- **Production Build**: `npm run build`
- **Type Checking**: `npm run type-check`
- **Deployment**: [Vercel Dashboard](https://vercel.com/dashboard)

### **🛠️ Technology Stack**
- **Frontend**: Next.js 15+, React 19, TypeScript 5
- **Styling**: Tailwind CSS 4 + shadcn/ui (planned)
- **Maps**: @vis.gl/react-maplibre v8 + MapLibre GL
- **State**: Zustand with React 19 concurrency
- **Backend**: Supabase (configured and ready)
- **Hosting**: Vercel (deployed)

### **📱 Geographic Scope**
- **Coverage**: Southern California (San Diego, Los Angeles, Inland Empire)
- **Boundaries**: Palmdale ↔ Ensenada, Yuma ↔ Santa Barbara
- **Map Tiles**: OpenStreetMap (100% free, no API tokens)
